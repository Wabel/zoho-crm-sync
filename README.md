Wabel's Zoho-CRM Synchronizer
=============================

Work in progress.

What is this?
-------------

This project is a set of tools to help you synchronize your database with
ZohoCRM records.
It is built on top of the [ZohoCRM ORM](https://github.com/Wabel/zoho-crm-orm).
